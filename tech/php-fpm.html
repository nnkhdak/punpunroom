PHP-FPM

Apache2.2にPHP5.3が入っているがPHP7.3をインストールしFPM化

  1. remiを使えるように sudo rpm -Uvh http://mirror.centos.org/centos/7/os/x86_64/Packages/scl-utils-20130529-19.el7.x86_64.rpm sudo yum install -y http://rpms.famillecollet.com/enterprise/remi-release-6.rpm sudo yum install -y yum-utils sudo yum-config-manager --enable remi
  2. php73のfpmをインストールし自動起動 sudo yum install -y php73-php-fpm sudo chkconfig php73-php-fpm on ← 自動起動の設定 chkconfig --list ← 自動起動されるか確認できる sudo service php73-php-fpm restart && sudo service httpd restart ← サービス再起動
  3. fastcgiをインストール cd /tmp/ wget http://ftp.tu-chemnitz.de/pub/linux/dag/redhat/el6/en/x86_64/rpmforge/RPMS/mod_fastcgi-2.4.6-2.el6.rf.x86_64.rpm sudo yum install -y mod_fastcgi-2.4.6-2.el6.rf.x86_64.rpm rpm -ql mod_fastcg ← インストールされたファイルが確認できる mkdir /var/www/fcgi-bin/ chmod +x /var/www/fcgi-bin/
  4. fastcgiの設定 sudo cp -p /etc/httpd/conf.d/fastcgi.conf /etc/httpd/conf.d/fastcgi.conf.original sudo sed -i -e 's/FastCgiWrapper On/FastCgiWrapper Off/' -e 's/FastCgiConfig/#FastCgiConfig/' -e '$aScriptAlias /fcgi-bin/ /var/www/fcgi-bin/' -e '$aAction php-fastcgi73 /fcgi-bin/php-fpm73' -e '$aFastCgiExternalServer /var/www/fcgi-bin/php-fpm73 -host 127.0.0.1:9000 -pass-header Authorization' /etc/httpd/conf.d/fastcgi.conf ↑「FastCgiWrapper On」を「Off」に変更 ↑「FastCgiConfig」の行をコメント化 ↑行末に以下を追加 ScriptAlias /fcgi-bin/ /var/www/fcgi-bin/ Action php-fastcgi73 /fcgi-bin/php-fpm73 FastCgiExternalServer /var/www/fcgi-bin/php-fpm73 -host 127.0.0.1:9000 -pass-header Authorization' /etc/httpd/conf.d/fastcgi.conf
  5. fastcgiの設定の余談
    /etc/opt/remi/php73/php-fpm.d/www.conf の「listen = 127.0.0.1:9000」と↑のFastCgiExternalServerの「-host」を一致させる。ここのポートを変えたりすればPHP-FPMの同居もできる
  6. FPMで起動させる対象を指定する(/etc/httpd/conf.d/php-handler.conf) <DirectoryMatch "/var/www/html/(aaa|bbb)"> <Files "*.php"> SetHandler php-fastcgi73 </Files> </DirectoryMatch> <DirectoryMatch "/var/www/html/ccc"> <Files "index.php"> SetHandler php-fastcgi73 </Files> </DirectoryMatch>
  7. phpinfoファイルを作成 echo '<?php phpinfo();' > /var/www/html/aaa/phpinfo.php
