PHP-FPM

AmazonLinux1にApache2.2にPHP5.3が入っているがPHP7.3をインストールしFPM化

  1. remiを使えるように yum repolist all ← 作業前に使用できないことを確認 sudo rpm -Uvh http://mirror.centos.org/centos/7/os/x86_64/Packages/scl-utils-20130529-19.el7.x86_64.rpm sudo yum install -y http://rpms.famillecollet.com/enterprise/remi-release-6.rpm sudo yum install -y yum-utils sudo yum-config-manager --enable remi yum repolist all ← 使えるようになったか確認
  2. php73のfpmをインストールし自動起動 sudo yum install -y php73-php php73-php-oci8 php73-php-pdo php73-php-mbstring php73-php-fpm sudo chkconfig php73-php-fpm on ← 自動起動の設定 chkconfig --list ← 自動起動されるか確認できる
  3. fastcgiをインストール cd /tmp/ wget http://ftp.tu-chemnitz.de/pub/linux/dag/redhat/el6/en/x86_64/rpmforge/RPMS/mod_fastcgi-2.4.6-2.el6.rf.x86_64.rpm sudo yum install -y mod_fastcgi-2.4.6-2.el6.rf.x86_64.rpm rpm -ql mod_fastcgi ← インストールされたファイルが確認できる mkdir /var/www/fcgi-bin/ chmod +x /var/www/fcgi-bin/
  4. fastcgiの設定 sudo cp -p /etc/httpd/conf.d/fastcgi.conf /etc/httpd/conf.d/fastcgi.conf.original sudo sed -i -e 's/FastCgiWrapper On/FastCgiWrapper Off/' -e 's/FastCgiConfig/#FastCgiConfig/' -e '$aScriptAlias /fcgi-bin/ /var/www/fcgi-bin/' -e '$aAction php-fastcgi73 /fcgi-bin/php-fpm73' -e '$aFastCgiExternalServer /var/www/fcgi-bin/php-fpm73 -host 127.0.0.1:9000 -pass-header Authorization' /etc/httpd/conf.d/fastcgi.conf ↑「FastCgiWrapper On」を「Off」に変更 ↑「FastCgiConfig」の行をコメント化 ↑行末に以下を追加 ScriptAlias /fcgi-bin/ /var/www/fcgi-bin/ Action php-fastcgi73 /fcgi-bin/php-fpm73 FastCgiExternalServer /var/www/fcgi-bin/php-fpm73 -host 127.0.0.1:9000 -pass-header Authorization' /etc/httpd/conf.d/fastcgi.conf diff /etc/httpd/conf.d/fastcgi.conf.original /etc/httpd/conf.d/fastcgi.conf sudo service php73-php-fpm restart && sudo service httpd restart ← サービス再起動
  5. fastcgiの設定の余談
    /etc/opt/remi/php73/php-fpm.d/www.conf の「listen = 127.0.0.1:9000」と↑のFastCgiExternalServerの「-host」を一致させる。ここのポートを変えたりすればPHP-FPMの同居もできる
  6. FPMで起動させる対象を指定する(/etc/httpd/conf.d/php-handler.conf) <DirectoryMatch "/var/www/html/(aaa|bbb)"> <Files "*.php"> SetHandler php-fastcgi73 </Files> </DirectoryMatch> <DirectoryMatch "/var/www/html/ccc"> <Files "index.php"> SetHandler php-fastcgi73 </Files> </DirectoryMatch>
  7. phpinfoファイルを作成 echo '<?php phpinfo();' > /var/www/html/aaa/phpinfo.php
  8. php73が起動するか確認 $ php73 -v PHP Warning: PHP Startup: Unable to load dynamic library 'oci8' (tried: /opt/remi/php73/root/usr/lib64/php/modules/oci8 (/opt/remi/php73/root/usr/lib64/php/modules/oci8: cannot open shared object file: No such file or directory), /opt/remi/php73/root/usr/lib64/php/modules/oci8.so (libclntsh.so.18.1: cannot open shared object file: No such file or directory)) in Unknown on line 0 PHP Warning: PHP Startup: Unable to load dynamic library 'pdo_oci' (tried: /opt/remi/php73/root/usr/lib64/php/modules/pdo_oci (/opt/remi/php73/root/usr/lib64/php/modules/pdo_oci: cannot open shared object file: No such file or directory), /opt/remi/php73/root/usr/lib64/php/modules/pdo_oci.so (libclntsh.so.18.1: cannot open shared object file: No such file or directory)) in Unknown on line 0 PHP 7.3.25 (cli) (built: Nov 24 2020 14:22:04) ( NTS ) Copyright (c) 1997-2018 The PHP Group Zend Engine v3.3.25, Copyright (c) 1998-2018 Zend Technologies $ libclntsh.so.18.1 と表示されていたのでOracle18を導入する
  9. zipをダウンロードし解凍、その後soファイルを/usr/lib64にコピー wget https://download.oracle.com/otn_software/linux/instantclient/185000/instantclient-basic-linux.x64-18.5.0.0.0dbru.zip wget https://download.oracle.com/otn_software/linux/instantclient/185000/instantclient-sqlplus-linux.x64-18.5.0.0.0dbru.zip wget https://download.oracle.com/otn_software/linux/instantclient/185000/instantclient-sdk-linux.x64-18.5.0.0.0dbru.zip unzip instantclient-sdk-linux.x64-18.5.0.0.0dbru.zip unzip instantclient-sqlplus-linux.x64-18.5.0.0.0dbru.zip unzip instantclient-basic-linux.x64-18.5.0.0.0dbru.zip cd instantclient_18_5/ la *so* rm libclntsh.so libocci.so sudo mv *so* /usr/lib64/ libclntsh.so libocci.so はシンボリックリンクのため削除
  10. php73が起動するか再確認 $ php73 -v PHP 7.3.25 (cli) (built: Nov 24 2020 14:22:04) ( NTS ) Copyright (c) 1997-2018 The PHP Group Zend Engine v3.3.25, Copyright (c) 1998-2018 Zend Technologies
  11. pdo確認 vi /tmp/pdo.php <?php $dbhost = 'ipaddress'; $dbport = '1521'; $dbname = 'database'; $dsn = 'oci:dbname='.$dbhost.':'.$dbport.'/'.$dbname; $user = 'user'; $pass = 'password'; try { $dbh = new PDO($dsn, $user, $pass); echo "OK"; } catch (PDOException $e) { echo $e->getMessage()."\n"; } echo "\n"; $ php /tmp/pdo.php OK $ php73 /tmp/pdo.php OK
