Visual Studio Code 関連

  • 日本語化
    1. 上部メニューの[View]->[Command Palette...]をクリック
    2. [configure]と入力し[Configure Display Language]をクリック
    3. [Install additional languages...]をクリック
    4. [Japanes Language Pack]の[日本語]の[Install]をクリック
  • WSLとのリモートデバッグ
    1. サイドバーにある[Extensions]をクリック(もしくはCtrl+Shift+X)
    2. [Search Extensions in Marketplace]のテキスト入力欄に[remote development]と入力し検索
    3. [Remote Development]の[Install]をクリック
    4. 同じく[php debug]をInstall
    5. 同じく[php intellisense]をInstall
  • WSLとのPHPのリモートデバッグ
    1. Xdebugをインストール
    2. pecl install xdebug
      xdebug.ini zend_extension=/usr/lib/php/20170718/xdebug.so xdebug.remote_autostart = 1 xdebug.remote_enable = 1 xdebug.remote_host = 127.0.0.1
  • WSLとのPHP5.4のリモートデバッグ
    1. Xdebugをインストール
    2. pecl install xdebug
      xdebug.ini zend_extension=/usr/lib/php/20170718/xdebug.so xdebug.remote_autostart = 1 xdebug.remote_enable = 1 xdebug.remote_host = 127.0.0.1
