git

  • ブランチ

    • ブランチをclone

      git clone -b ブランチ名 --single-branch https://github.com/リポジトリ名 .

    • ブランチにcommitとpush

      git add -A git commit git push origin ブランチ名

    • ローカルの変更を全て破棄する

      cd /var/www/ && git checkout .

    • ブランチの変更

      cd /var/www/ && git fetch && git checkout branchname

    • ローカルの状態をキレイにする

      cd /var/www/ && git clean -df

    • 現在のブランチ名を表示

      cd /var/www/ && git branch --contains=HEAD

    • 実行権限の付与、除外

      git update-index --add --chmod=+x filename git update-index --add --chmod=-x filename

    • git add を取り消す

      git reset HEAD
  • その他

    • 空ディレクトリに.gitkeepを置く

      cd 置きたいディレクトリのルート find . -type d -name .git -prune -o -type d -empty -exec touch {}/.gitkeep \;
