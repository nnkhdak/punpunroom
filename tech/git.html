git

  • ブランチ

    • タグを打つ

      git tag -a tagname commit git push origin tagname

    • ローカルブランチを削除

      git branch -d ブランチ名

    • リモート追跡ブランチを全削除

      git fetch -p git remote prune origin

    • 直前のコミットを取り消す

      git reset --soft HEAD^

    • clone

      git clone https://github.com/リポジトリ名

    • ブランチをclone

      git clone -b ブランチ名 --single-branch https://github.com/リポジトリ名 .

    • ブランチを作成しつつ変更＆push

      git checkout -b ブランチ名 git push --set-upstream origin ブランチ名

    • commitとpush

      git add -A git commit git push

    • ブランチの変更

      git fetch && git checkout branchname

    • ローカルの状態をキレイにする

      git clean -df

    • 現在のブランチ名を表示

      git branch --contains=HEAD

    • 実行権限の付与、除外

      git update-index --add --chmod=+x filename git update-index --add --chmod=-x filename

    • git add を取り消す

      git reset HEAD

    • ブランチ同士の比較

      git diff ブランチA..ブランチB
  • その他

    • 空ディレクトリに.gitkeepを置く

      cd 置きたいディレクトリのルート find . -type d -name .git -prune -o -type d -empty -exec touch {}/.gitkeep \;
