  • Windows10でoggファイルの操作が遅い
    1. WindowsPowerShell (管理者)を起動
    2. Get-AppxPackage *Microsoft.WebMediaExtensions* | remove-appxpackage
  • Linuxのホスト名を変えたい
    sudo vim /etc/hosts 127.0.0.1 localhost localhost.localdomain REPLACE_WITH_NEW_HOST
    sudo vim /etc/sysconfig/network HOSTNAME=REPLACE_WITH_NEW_HOST
    上記を対応後、サーバー再起動が必要
  • Linux上でjsonを整形したい cat 整形前ファイル名 | python -m json.tool
  • GitHubのWebページでタブ幅 URL の末尾に ?ts=4
