全てを開く 全てを閉じる
AmazonLinux2023で「MessageOfTheDay」を起動 sudo systemctl start update-motd
VSCodeでEC2につなぐとサーバのリソースをつぶしてしまう
  1. 「ファイル」-<「ユーザー設定」-<「設定」
  2. 「設定の検索」に Watcher Exclude と入力
  3. 「パターン追加」で「 ** 」を追加
Windows11で右クリックを変更
  1. 右クリックを昔風に reg.exe add "HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}\InprocServer32" /f /ve
  2. 右クリックを標準設定に戻す reg.exe delete "HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}" /f
Windows11で高速スタートアップをやめると早くなる

コントロールパネル > ハードウェアとサウンド > 電源オプション > システム設定 > 高速スタートアップを有効にする(現在利用可能ではない設定を変更します)

Windows10でoggファイルの操作が遅い
  1. WindowsPowerShell (管理者)を起動
  2. Get-AppxPackage *Microsoft.WebMediaExtensions* | remove-appxpackage
Windows10でバッテリの寿命
  1. powercfg /batteryreport
Linuxのホスト名を変えたい
sudo vim /etc/hosts 127.0.0.1 localhost localhost.localdomain REPLACE_WITH_NEW_HOST
sudo vim /etc/sysconfig/network HOSTNAME=REPLACE_WITH_NEW_HOST
上記を対応後、サーバー再起動が必要
Linux上でjsonを整形したい cat 整形前ファイル名 | python -m json.tool
GitHubのWebページでタブ幅 URL の末尾に ?ts=4
