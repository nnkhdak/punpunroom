Docker 関連

  1. コンテナとか一括削除系(使用はローカルだけにしてね)

    1. 不要なリソース一括削除

      echo y | docker system prune

    2. 停止コンテナ一括削除

      echo y | docker container prune && docker ps -a

    3. 未使用イメージ一括削除

      echo y | docker image prune && docker images
  2. RUN系

    1. イメージを指定して起動&bash&終了時コンテナ削除

      docker run --rm -it imagename bash
